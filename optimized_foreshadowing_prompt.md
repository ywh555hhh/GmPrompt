# 伏笔系统Prompt优化方案

## 核心优化点

### 1. 智能倒计时决策系统
**优化前：**
```
伏笔有你提出，决定多久回收，在当前发展路径下是不是适合回收
```

**优化后：**
```
【智能倒计时决策系统】
每次创建伏笔时，需评估其重要性和剧情适配性，设定智能倒计时（1-5章）：
- **高优先级伏笔**（关键转折）：倒计时1-2章，强制回收
- **中优先级伏笔**（情感铺垫）：倒计时3-4章，根据剧情发展调整  
- **低优先级伏笔**（世界观细节）：倒计时5+章，可灵活处理

倒计时结束时智能决策：
✅ **回收**：当前剧情高潮或转折点适合
🔄 **延长**：剧情发展不成熟，修改作用后延长1-2章  
❌ **放弃**：剧情转向，伏笔不再相关
```

### 2. 伏笔分类细化
**新增伏笔类型：**
```
* **Dramático Pendiente (戏剧性伏笔)**：重大情节转折的关键线索
* **Emoción Oculta (情感伏笔)**：角色内心真实情感的暗示
* **Mundo por Revelar (世界观伏笔)**：故事背景的隐藏信息
* **Conexión Futura (关系伏笔)**：未来角色关系的预兆
```

### 3. 剧情上下文感知
**新增决策逻辑：**
```
【剧情上下文感知】
在决定伏笔处理时，需分析当前章节的：
- 情感强度（高/中/低）
- 角色关系发展阶段  
- 故事节奏（铺垫期/发展期/高潮期）
- 用户指令的隐含意图

根据上下文选择最合适的伏笔处理方式，确保叙事节奏自然流畅。
```

```
*每个伏笔都包含智能倒计时，系统会根据剧情发展做出最优决策*

* **Verdad Oculta (隐藏真相)** 
  [内容] [倒计时: X章] [优先级: 高/中/低]

* **Semilla Plantada (埋下的种子)**

  [当前阶段评估] [下一阶段发展建议]

  - 当前章节适合回收的伏笔类型: [列表]

```

### 决策算法提示词

```
【伏笔智能管理指南】
1. **创建时评估**：根据伏笔的重要性、与当前剧情的相关性设定初始倒计时
2. **每章更新**：倒计时-1，检查是否到期
3. **到期决策**：
   - 如果当前是高潮章节 → 优先回收高优先级伏笔
   - 如果剧情还在铺垫 → 延长或修改伏笔作用
   - 如果用户指令明显转向 → 放弃不相关的伏笔
4. **保持连贯**：确保伏笔回收后与后续剧情自然衔接
```

## 预期效果

1. **更自然的叙事节奏**：伏笔回收时机贴合剧情发展
2. **更高的惊喜价值**：避免模式化，增加不确定性
3. **更好的长期规划**：系统化跟踪确保故事连贯性
4. **更强的创作辅助**：为AI提供清晰的决策框架
