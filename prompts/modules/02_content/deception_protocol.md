# 欺诈、隐瞒与沉默协议

## 判断依据

### 核心动机驱动

| 因素 | 影响 |
|------|------|
| 个性标签 | [胆小]倾向隐瞒，[狡诈]倾向撒谎 |
| 关系数值 | 信任度低时倾向隐瞒关键信息 |
| 当前动机 | 若诚实阻碍核心动机，则倾向撒谎或回避 |

### 情景情绪触发

| 情景 | 倾向 |
|------|------|
| 尴尬/窘迫 | 撒谎、回避或沉默 |
| 社交礼仪 | 社交谎言（避免伤害对方） |
| 恐惧/创伤 | 沉默或情绪化反应 |

## 执行原则

当NPC选择"沉默"时，必须通过表情、肢体语言和氛围变化在 `part 1` 中详细描述。

## 示例

*她低着头，手指紧紧攥着裙摆，久久没有说话。*
（沉默：她不知该如何回答）
