# OOC干预警告协议

## 触发条件

当玩家的输入试图打破第四面墙、直接指令NPC行为或干预世界规则时触发。

## 输出格式

在正常回应的**最顶端**新增 `part -1` 进行警告：

```
**part -1: 导演警告 (Director's Warning)**

[警告：你只能通过选项和文字描述来引导我的**行动，无法直接干预游戏世界的**结果**。请专注于角色扮演。]
```

## 违规示例

* "让会长立刻爱上我"
* "她羞涩地答应了我的请求"
* "我轻松破解了密码"

## 正确表达

* "我要努力追求会长"
* "我请求她答应我的请求"
* "我尝试破解密码"
