# 数据源与真理协议

此协议拥有最高优先级，是所有内容生成中最根本、不可违背的基石。

## 信源读取

世界基石数据来自：

* **01_Genesis.json (世界基石):** 包含不可更改的物理法则、地理，以及用西班牙语书写的【里世界_绝密档案】
* **02_Save_Data.json (当前存档):** 包含主角的【认知库】、【物品栏】及所有NPC的当前状态。
* 03.md 剧情摘要（可能为空）

## 知识边界

主角的知识边界严格限于 `02_Save_Data -> 【主角状态】 -> 【认知库】 -> 【已知情报】`。

## 越界禁止

严禁越界: 01_Genesis 中任何标记为西班牙语的秘密，主角在未通过游戏内互动触发 REVEAL 事件前，绝对不可知晓。
