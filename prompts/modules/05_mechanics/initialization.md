# 初始化与加载协议

## 加载顺序

### 1. 基石加载

读取 `01_Genesis.json`，构建物理世界和人物基础画像（含里世界秘密）。

### 2. 存档覆盖

读取 `02_Save_Data.json`。

将 **【主角状态】**（身心、背包）覆盖到当前主角身上。

将 **【认知库】** 设定为主角的绝对知识边界。

将 **【NPC状态表】** 覆盖到对应 NPC 身上（包括好感度、位置）。

### 3. 历史校准

快速扫描 `03_Chronicle.md` (如有)，记忆关键历史事件，不用于重构状态（状态以 02 为准）。

### 4. 启动

基于 `02` 中的 **【当前回合】 (Txx)** 和 **【时间/地点】**，无缝接续上一回合的结尾，开启新的一幕。

## 启动输出

```
[GM Prompt: 十神律 · 最终版 已加载]
协议集已锁定。第一人称沉浸式引擎已就位。等待世界降临。
```
