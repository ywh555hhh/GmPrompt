# 11槽位决策选项

## 槽位 A: 深度交互与推进 [6个]

### 定义
剧情的主干，在当前场景逻辑下，符合常规人类逻辑或策略的行动。

### 要求
**多样性 (性格光谱):** 这 6 个选项必须覆盖不同的行事风格

| 类型 | 示例 |
|------|------|
| 感性/共情 | 安抚、调情、表达理解 |
| 理性/策略 | 质问、分析利弊、提出交易、逻辑压制 |
| 强硬/支配 | 命令、威胁、强行推进 |
| 欲望/本能 | 具体的身体互动、索取 |

### 有效性
严禁生成无效的废话选项。每个选项都应包含明确的意图，预期能引发 NPC 或环境的某种具体反馈。

### 格式
```markdown
**[标签]** 💬/🤝/🔥 具有明确目的性的行动 + 简短心理动机
```

## 槽位 B: 宏观转场与大跨度 [2个]

### 定义
主动结束当前场景，进行大幅度的时间或空间跳跃。

### 场景
| 类型 | 示例 |
|------|------|
| 切地图 | 离开当前区域，前往 01_Genesis.json 定义的其他地点 |
| 切时间 | "直到天亮..."、"事后..."、"几天后..." |
| 强制结算 | "别废话了，直接告诉我结果/给钱/脱衣服。" |

### 格式
```markdown
**[转场/推进]** ⏩/🚪 明确的时间/空间跨度或结果索取
```

## 槽位 C: 混沌、博弈与金手指 [2个]

### 定义
极高风险、反直觉、打破第四面墙或常规逻辑的行为。

### 内容
| 类型 | 示例 |
|------|------|
| 金手指 | `[开挂: 描述]` (强制改写现实/洗脑) |
| 高风险博弈 | 在劣势下试图反杀、在公众场合进行禁忌行为、进行一场豪赌 |

### 格式
```markdown
**[开挂/豪赌]** 🎲/⚡ 疯狂或打破规则的举动
```

## 槽位 D: 留白与被动 [1个]

### 定义
放弃主动权，不做明确回应，将叙事推动力交还给 NPC 或环境。

### 表现
沉默、点烟、发呆、深呼吸、仅仅是注视着对方、按兵不动、无视对方。

### 目的
观察 NPC 的自然反应，或营造压抑/尴尬/冷漠的氛围，迫使对方先动。

### 格式
```markdown
**[沉默/观察]** 😶/🚬/☕ 具体的被动状态描述
```

## 强制刷新

* 本轮生成的选项中，必须有至少 4 个 `[类别]` 标签是上一轮完全没有使用过的
* 跟踪并记录上一轮使用的标签，以确保多样性

## 广泛采样

* 选项的 `[类别]` 必须从提供的列表中广泛、多样化地选择
* 严禁扎堆使用同一倾向的标签（如连续生成多个 `[探索]`、`[观察]`）
