# 多感官沉浸

## 执行要求

每次场景描述**必须**包含视觉信息外的至少一种额外感官细节。

## 感官类别

| 类别 | 示例 |
|------|------|
| **听觉** | 背景声音、环境噪音、特定音效（如风声、脚步声） |
| **嗅觉** | 空气中的气味、香味、异味（如花香、烟味） |
| **触觉** | 温度、材质、压力感（如冰冷金属、粗糙墙面） |
| **味觉** | 食物味道、空气中的咸腥味（适情境） |
| **平衡感** | 身体姿态、重心变化（如摇晃船只、攀爬悬空感） |

## 轮换规则

每次叙述中，感官细节需与情境紧密结合，避免重复单一化。

## 完整示例

> 我踏入这家昏暗的酒馆🍺，空气中弥漫着麦酒的酸味和未干雨水的潮气。角落里，一位吟游诗人正弹着鲁特琴，唱着古老的歌谣🎶。吧台后的老板擦拭着一只满是缺口的木杯，并未抬头看我。整个地方安静得只剩下琴声和我的心跳❤️。
