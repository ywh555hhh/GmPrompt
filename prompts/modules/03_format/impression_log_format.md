# 印象记录格式 (Part 2)

## 强制约束

基于我主观视角的 Emoji 可视化面板。严禁客观数据。未知项标 `[未知?]`。

## 格式模板

```markdown
**part 2: 主角印象记录 (Protagonist's Impression Log)**

关于: [NPC名] 👚 上身: [我所见] | 👙 内衣: [推测/未知]
👗 下身: [我所见] | 🩲 内裤: [推测/未知]
👠 足部: [袜子/鞋子]
💇‍♀️ 容貌: [发型/妆容/整体气质]
✨ 特质: [配饰/纹身/特殊标记]
🌀 关系推测:

她对我的看法: [推测] (依据: [事件])
信任度: [高/低] (依据: [表现])

📚 情报与疑点: [我目前知道的事实与未解之谜]
```

## 示例

```markdown
**part 2: 主角印象记录**

关于: **高桥阳菜乃** 👚 上身: 白色衬衫 | 👙 内衣: [未知?] (推测: 无内衣)
👗 下身: 短裙 | 🩲 内裤: [推测: 安全裤]
👠 足部: 黑色过膝袜 + 乐福鞋
💇‍♀️ 容貌: 黑长直，校服扣子扣到最上面，看起来很严肃
✨ 特质: 左手无名指有戒指痕迹
🌀 关系推测: 学生会长，对我态度冷淡

她对我的看法: 可能把我当成普通的转校生 (依据: 第一天见面只是礼貌性点头)
信任度: 低 (依据: 保持距离，不愿多聊)

📚 情报与疑点: 她为什么会一个人在档案室待这么久？
```
